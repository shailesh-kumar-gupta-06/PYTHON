# 🧮 Advanced BMI Calculator (Python GUI App)

A fully functional **BMI (Body Mass Index) Calculator** built using Python with a graphical user interface, database integration, and data visualization.

This project goes beyond a simple calculator by allowing users to **store, track, and analyze BMI history over time**.

---

## 🚀 Features

* ✅ User-friendly GUI using Tkinter
* ✅ Accurate BMI calculation
* ✅ BMI categorization (Underweight, Normal, Overweight, Obese)
* ✅ Multi-user support (name-based tracking)
* ✅ Data stored using SQLite database
* ✅ BMI history tracking with timestamps
* ✅ Graph visualization of BMI trends using Matplotlib
* ✅ Input validation and error handling

---

## 📊 BMI Formula

BMI is calculated using the formula:

BMI = weight (kg) / (height (m))²

---

## 🛠️ Tech Stack

* **Python 3**
* **Tkinter** (GUI)
* **SQLite3** (Database)
* **Matplotlib** (Data Visualization)

---

## 💡 How to Use

1. Enter your **Name**
2. Input your **Weight (kg)**
3. Input your **Height (meters)**
4. Click **"Calculate BMI"**
5. View:

   * Your BMI result
   * Health category

### Additional Options:

* 📜 **Show History** → View past BMI records
* 📈 **Show Graph** → Visualize BMI trends over time
