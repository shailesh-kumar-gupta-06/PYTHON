# 🎙️ Voice Assistant (Jarvis)

## 📌 Description

A Python-based voice assistant that listens to user commands and performs tasks like telling time, opening websites, and responding with speech.

## 🚀 Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech response
* 🕒 Tells current time and date
* 🌐 Opens websites like Google
* 🤖 Simple conversational responses

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3
* datetime
* webbrowser

## ⚙️ Installation

```bash
git clone https://github.com/shailesh-kumar-gupta-06/voice-assistant.git
cd voice-assistant
pip install -r requirements.txt
```

## ▶️ Usage

```bash
python main.py
```
